# Hyprland Configuration

Modular Hyprland configuration for blxshell with Quickshell integration.

## Structure

```
hypr/
├── hyprland.conf          # Main entry point
├── autostart.conf         # Startup applications
├── monitor.conf           # Display configuration
├── variables.conf         # Custom variables
├── colors.conf            # Color scheme (generated by matugen)
├── current_wall.conf      # Current wallpaper path
├── hypridle.conf          # Idle daemon config
├── hyprlock.conf          # Lock screen config
├── hyprpaper.conf         # Wallpaper daemon config
│
├── visual/                # Appearance settings
│   ├── general.conf       # Gaps, borders, layout
│   ├── decoration.conf    # Blur, shadows, rounding
│   ├── layout.conf        # Layout-specific settings
│   ├── o2-vertical.conf   # Vertical workspace animations (active)
│   └── o2-horizontal.conf # Horizontal workspace animations (alt)
│
├── keyboard/              # Input configuration
│   ├── binds.conf         # Keybindings
│   ├── gestures.conf      # Touchpad gestures
│   └── kblayout.conf      # Keyboard layout
│
├── rules/                 # Window/layer rules
│   ├── windowrules.conf   # Per-app window rules
│   └── layerrules.conf    # Layer shell rules
│
└── scripts/               # Utility scripts
    ├── audio.sh           # Audio control helper
    └── f_wallpaper.sh     # Fallback wallpaper (legacy)
```

## Key Features

- **Modular Configuration**: Organized into logical sections
- **Material Design 3**: Color scheme via matugen integration
- **Quickshell Integration**: Custom shell UI with global keybinds
- **Animation Variants**: Switch between vertical/horizontal workspace animations
- **Dynamic Theming**: Colors auto-generated from wallpaper

## Configuration Files

### Main Files

**`hyprland.conf`**
- Main entry point that sources all other configs
- Switch animation styles here (line 8-9)

**`autostart.conf`**
- Startup applications
- Currently launches: Quickshell (qs), dex autostart

**`monitor.conf`**
- Display configuration
- Format: `monitor = name, resolution@refresh, position, scale`
- Example: `monitor = HDMI-A-1, 1920x1080@179.82, 0x0, 1.00`

**`variables.conf`**
- Custom variables (e.g., `$mainMod`, `$terminal`)
- Reusable values across configs

### Visual Configuration

**`visual/general.conf`**
- Gaps: `gaps_in = 3`, `gaps_out = 5`
- Borders: `border_size = 1`
- Border colors from matugen
- Layout: `layout = scrolling`

**`visual/decoration.conf`**
- Blur settings
- Shadow configuration
- Corner rounding

**`visual/layout.conf`**
- Layout-specific settings (dwindle, master, etc.)

**Animation Variants:**
- `o2-vertical.conf` - Vertical workspace transitions (default)
- `o2-horizontal.conf` - Horizontal workspace transitions

To switch: Edit `hyprland.conf` line 8-9

### Input Configuration

**`keyboard/binds.conf`**
Key keybindings:
- `Super + Return` - Terminal
- `Super + Q` - Kill window
- `Super + Space` - Launcher (Quickshell)
- `Super + S` - Settings (Quickshell)
- `Super + W` - Random wallpaper
- `Super + 1-9` - Switch workspace
- `Super + Shift + 1-9` - Move window to workspace
- `Print` - Screenshot
- Media keys - Audio/brightness control

**`keyboard/gestures.conf`**
- Touchpad gesture configuration

**`keyboard/kblayout.conf`**
- Keyboard layout settings
- Repeat rate/delay

### Rules

**`rules/windowrules.conf`**
- Per-application window rules
- Opacity, floating, workspace assignments

**`rules/layerrules.conf`**
- Layer shell rules for panels, notifications, etc.

## Theming

Colors are managed by [matugen](https://github.com/InioX/matugen):

1. **Generated File**: `colors.conf`
2. **Source**: Current wallpaper
3. **Scheme**: Material Design 3 (scheme-content)

### Color Variables

Used throughout the config:
```
$primary
$on_primary
$surface_container
$background
$foreground
# ... and more
```

### Regenerate Colors

```bash
# Matugen will auto-generate on wallpaper change
# Manual regeneration:
matugen image /path/to/wallpaper.png
```

## Customization

### Change Monitor Configuration

Edit `monitor.conf`:
```conf
# Single monitor
monitor = DP-1, 1920x1080@60, 0x0, 1

# Dual monitors
monitor = DP-1, 2560x1440@144, 0x0, 1
monitor = HDMI-A-1, 1920x1080@60, 2560x0, 1

# Auto
monitor = , preferred, auto, 1
```

### Change Animation Style

Edit `hyprland.conf` line 8-9:
```conf
# Vertical (default)
source = ~/.config/hypr/visual/o2-vertical.conf

# Horizontal
source = ~/.config/hypr/visual/o2-horizontal.conf
```

### Modify Gaps/Borders

Edit `visual/general.conf`:
```conf
gaps_in = 3      # Inner gaps
gaps_out = 5     # Outer gaps
border_size = 1  # Border thickness
```

### Add Keybinds

Edit `keyboard/binds.conf`:
```conf
bind = $mainMod, X, exec, your-command
```

### Add Window Rules

Edit `rules/windowrules.conf`:
```conf
windowrule = float, ^(app-name)$
windowrule = workspace 2 silent, ^(app-name)$
```

## Troubleshooting

### Quickshell not launching
```bash
# Check if quickshell is installed
which qs

# View logs
journalctl --user -u quickshell
```

### Colors not updating
```bash
# Manually regenerate
matugen image ~/.config/hypr/current_wall.conf

# Check if colors.conf exists
ls -la ~/.config/hypr/colors.conf
```

### Monitor not detected
```bash
# List available monitors
hyprctl monitors

# Add to monitor.conf
monitor = YOUR-MONITOR, preferred, auto, 1
```

### Keybinds not working
```bash
# List all binds
hyprctl binds

# Check for conflicts
grep -r "bind.*YOUR_KEY" ~/.config/hypr/
```

## Performance Tips

1. **Disable blur** if on low-end hardware:
   ```conf
   # visual/decoration.conf
   blur {
       enabled = false
   }
   ```

2. **Reduce animations**:
   ```conf
   animations {
       enabled = false
   }
   ```

3. **Lower refresh rate** if needed:
   ```conf
   monitor = DP-1, 1920x1080@60, 0x0, 1
   ```
